BEM
  • BEM (Block, Element, Modifier) is a component-based approach to web development.
  • The idea behind it is to divide the user interface into independent blocks
  • BEM is a popular naming convention for classes in HTML and CSS
  • it allows reuse of existing code without copying and pasting.
  • syntax:: .block-name__element-name--modifier-name {}
